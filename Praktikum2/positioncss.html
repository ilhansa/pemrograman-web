position : static;

position static digunakan untuk elemen default. elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi oleh properti seperti top, bottom, left, dan right

elemen ini diaatur menggunakan posisi static




position : relative;

sebuah elemen html yang menggunakan position relative akan terletak pada posisi normal. mengatur properti atas, kanan, kiri, bawah dari elemen yang diposisikan dengan "relatif" akan membuatnya jauh dari posisi normal

elemen ini diatur menggunakan posisi relatif




position : fixed

sebuah elemen html yang seting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika website discroll

elemen ini diatur menggunakan posisi fixed












position absolute

elemen html yang diposisikan aboslute akan diposisikan relatif dengan elemen lain yang mendahuluinya yang terdekat, bukan relatif secara normal

elemen ini diatur menggunakan position relative
elemen ini diatur menggunakan position absolute








position : sticky

cobalah untuk scroll untuk mengetahui bagaimana posisi sticky bekerja

catatan : IE/Edge 15 dan versi yang lebih lama tidak mendukung sticky

hola aku adalah sticky

dalam contoh ini, elemem sticky menempel pada posisi paling atas halaman (top:0), ketika mencapai posisi scrollnya
